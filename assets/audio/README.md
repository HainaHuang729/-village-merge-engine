# Audio Placeholders

This directory is reserved for sound effects. No placeholder audio files are generated in this phase.

Reserved file names:

- `sfx_merge.mp3`
- `sfx_click.mp3`
- `sfx_build.mp3`
- `sfx_truck.mp3`
- `sfx_coin.mp3`

All audio must be referenced through `AssetManager`, not hard-coded paths.
